Programación con Python
===================================

Aprenderás a programar mientras das vida a tu videojuego, desde la idea inicial hasta los efectos de sonido y la música.

**¿Qué aprenderás?**

* A usar variables y a crear funciones como si fueran los ladrillos de tu juego,
* A crear estructuras de control para que los personajes se muevan como tú quieras,
* A manejar eventos para que el juego responda a las acciones del jugador,
* A ponerle sonido y efectos para que tu juego sea una experiencia inmersiva y emocionante.
